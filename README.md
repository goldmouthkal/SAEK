# Student Manager – Python 2.7.10
Mini εφαρμογή κονσόλας σε Python 2.7.10 για εξάσκηση στη συνεργασία με Git
& GitHub.
## Περιγραφή
Η εφαρμογή επιτρέπει:
- προσθήκη μαθητή (όνομα, βαθμός)
- εμφάνιση λίστας μαθητών
- υπολογισμό μέσου όρου βαθμών
- εμφάνιση καλύτερου μαθητή
Όλα τα δεδομένα αποθηκεύονται προσωρινά στη μνήμη.
